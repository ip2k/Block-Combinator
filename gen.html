This is a custom blocklist generator that allows you to combine multiple blocklists.
It supports P2P formatted blocklists in both gzipped and non-gzipped formats.
The main motivation behind this is torrent clients that limit you to a single auto-updating blocklist URL.
You can find P2P-format blocklists on Google, so don't ask for URLs.

Usage

  1. Paste URLs of gzip compressed P2P format blocklists here, one per line, then click "Generate". Stretch the box as needed so that each URL is on a separate line.
  2. This app will then fetch all the blocklists, ungzip them, combine them, re-gzip them, and present you with a download.
  3. All necessary parameters are encoded in the URL, so if you re-download the generated URL, it'll fetch all new updated blocklists
  4. Due to the above, you can set the "Enable blocklist" URL in Transmission to the URL of the next page, and enable automatic updates
